# Biji Toolkit for Corona SDK

A simple toolkit for Corona SDK, especially for building flat UI business app.

## Features

### Controls
1. Flat Button
2. Header
3. Sliding Menu
4. Notification

### Assets
1. Flat Colors (flatuicolors.com)
2. Fonts (Roboto & Fabrica)
3. Icons (icons8.com)

### Tools
1. Logger
2. Controls
